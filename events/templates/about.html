About Events Calendar

This is a simple and elegant event management application built using the Django framework. It allows users to easily schedule, track, and manage their personal events with modern date/time picker functionality.

Key Features:

  • User Authentication (Sign Up/Login)
  • Event Creation, Editing, and Deletion (CRUD)
  • Search functionality for events
  • Modern, responsive design
